# CarbonLens Next Steps

## Immediate Improvements After MVP

- Expand Scope 3 categories beyond the initial demo set
- Add better validation coverage for activity units and reporting periods
- Improve report formatting and export options
- Add richer issue assignment and status flows

## Product Expansion

- Add scenario planning for emissions reduction pathways
- Add benchmarking by facility or category
- Support multiple reporting periods and comparisons
- Introduce targets, alerts, and action plans

## Data And Integration Roadmap

- ERP integration for procurement and operational data
- Utility and meter ingestion integrations
- Supplier onboarding workflows and reminders
- External emission factor library management

## Compliance And Governance Roadmap

- Map outputs to major ESG and sustainability disclosure frameworks
- Strengthen audit trails and data lineage views
- Add approval workflows and formal escalation paths
- Add role-based permissions and review states

## Analytics Roadmap

- Forecasting and trend projections
- Cost-to-carbon correlation
- Reduction opportunity scoring
- Scenario modeling with what-if assumptions

## Enterprise Readiness Roadmap

- Multi-tenant architecture
- Organization-level customization
- Stronger authentication and access control
- Enterprise-grade reporting templates
- API integrations for ecosystem partners

## Long-Term Vision

CarbonLens can evolve from a hackathon MVP into a practical sustainability operations platform for organizations that need an affordable, trustworthy, and action-oriented system for ESG performance and GHG monitoring.
